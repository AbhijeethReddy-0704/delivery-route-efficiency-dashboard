## Capture Tips
- Run app: `streamlit run streamlit_app.py`
- Use full-width browser window (recommended 1440px+).
- Keep sidebar visible in at least the first screenshot.
- Capture each page after filters are set to a representative 12-month view.
- Avoid blank states and warning messages in screenshots.
- Use PNG for crisp rendering on GitHub.

## What to Show per Page
- Executive Overview: KPI cards + SLA opportunity hero + donut + trend lines
- Route Bottlenecks: leaderboard + top 10 bottlenecks + heatmap + bubble chart
- Delivery Trends: monthly area + SLA line + weekday grouped bars + rolling late trend
- Failure Analysis: reason donut + zone stacked bars + route-reason heatmap
- Driver/Vehicle: top-bottom driver table + productivity scatter + vehicle charts
- Cost/Efficiency: cost trend + cost/mile + distance-cost scatter + ranking
- Data Quality: pass/fail cards + validation summary + freshness note
